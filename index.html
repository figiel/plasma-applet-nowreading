Now Reading Applet

A plasma applet showing recently open documents from Okular with progress bar.

Installation

Build dependencies:

sudo apt-get build-dep kde-workspace
sudo apt-get install libpoppler-qt4-dev

Clone needed repositories:

git clone git://github.com/figiel/plasma-engine-nowreading.git
git clone git://github.com/figiel/plasma-applet-nowreading.git

Build and install to /usr/local:

mkdir plasma-{engine,applet}-nowreading/build
cd plasma-engine-nowreading/build
cmake ../
make
sudo make install
cd ../../plasma-applet-nowreading/build
cmake ../
make
sudo make install

Adjust KDEDIRS to include /usr/local, e.g. by doing this:

echo -en '#!/bin/bash\nexport KDEDIRS=/usr/local\n' > ~/.kde/env/plasmoids.sh
chmod a+x ~/.kde/env/plasmoids.sh

Limitations

At the moment only PDF documents are shown.
